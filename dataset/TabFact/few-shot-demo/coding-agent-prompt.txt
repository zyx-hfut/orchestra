Intermediate table:
[HEAD]: date|opponent|venue|result|attendance

Output:
Python: ```def is_tie(result):
    scores = result.split(' - ')
    return int(scores[0]) == int(scores[1])

DF['is_tie'] = DF['result'].apply(is_tie)```.
