Instruction:
Extract numerical values of prominence in feet and add as a new column 'prominence_ft'.

Output:
Python: ```def get_ft(s):
    import re
    return float(re.search("(.*?) ft", s).group(1).replace(',', ''))
DF['prominence_ft'] = DF.apply(lambda x: get_ft(x['prominence']), axis=1)
```.
